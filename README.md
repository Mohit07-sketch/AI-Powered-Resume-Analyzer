AI-Powered Resume Analyzer

This project is a simple web application built with Streamlit that analyzes resumes and provides AI-based feedback to help improve them. Users can upload their resume in PDF format, and the application extracts the text using PyPDF2 before sending it for AI analysis.
The system evaluates the resume based on a selected job role, providing tailored suggestions and a professional critique. It highlights areas that could be improved, suggests better wording, and helps users understand how well their resume fits a particular position.
The goal of this project is to demonstrate how AI can be integrated into a web application to automate resume evaluation and provide meaningful feedback for job seekers. Basic validation and error handling are also implemented to ensure the app processes uploaded files reliably.

Key Features:

1.Clean and interactive Streamlit web interface
2.PDF resume parsing using PyPDF2
3.AI-powered resume analysis with OpenAI
4.Job-role specific feedback
5.Professional resume critique generation
6.File validation and error handling
